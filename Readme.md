# Flask Calculator

A simple calculator web application built using Python Flask, HTML, and CSS.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Clean and responsive user interface

## Technologies Used

* Python
* Flask
* HTML
* CSS

## How to Run

1. Install Flask:
   pip install flask

2. Run the application:
   python app.py

3. Open:
   http://127.0.0.1:5000
